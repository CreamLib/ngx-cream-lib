Setting Up Your Local Environment

Remember that, in order to install Angular on your local system, you'll need Node.js and to install an npm package manager with the command npm install npm -g

CreamNg uses Angular version 15, so be sure to have the latest version of Angular-Cli:

npm install -g angular-cli

Importing CreamNg into Your Project

Once your project created, import the library via npm (run the command on the root of your project):

npm install ngx-cream-lib

Now, include the CreamNg module in your app.module.ts by importing the module and declaring it on the "imports" list: 

import { NgxCreamLibModule } from 'ngx-cream-lib';

...
imports: [NgxCreamLibModule]
