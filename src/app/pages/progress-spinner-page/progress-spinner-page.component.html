General Component

Giving a template variable when calling the component allows you to trigger or stop the spinner. 

<c3m-progress-spinner #process1>
</c3m-progress-spinner>

EXAMPLE TO TRIGGER THE SPINNER

<button (click)="insertContent(process1); process1.ToggleOpen($event); stopProgress(process1)">
  Open spinner
</button>

insertContent(spinner: ProgressSpinnerComponent) {
  spinner.progressText = "Loading...";
}

// Demo for progress spinner
// Sends signal to stop progress and close spinner
stopProgress(el: any) {
  setTimeout(function () {
    el.isOpen = false;
    el.progressText = "";
  }, 3000);
}

The text to be announced by screen readers must be injected dynamically to the component.

Properties applicable to the component c3m-progress-spinner
