Properties and Events

Properties applicable to the component c3m-tabs
Name Type Default Description
label * string -- Describes the nature of the group of tabs
[handleScroll] boolean false Allows to add buttons to handle the scroll of tabs
startIndexActiv number 0 Outputs index of deleted tab
discrete (class) -- Simplifies tabs design style
Properties applicable to the component c3m-tab-panel
Name Type Default Description
[isActive] * boolean -- Sets default selected tab
tabTitle * string -- Label for the tab
[deletable] boolean false Adds delete button to tab
[draggable] boolean false Makes tab draggable along tabs list
[adaptable] boolean false Allows tab container to have fixed height and vertical scroll
svgName string -- SVG image ID (from index svg sprite)
svgTitle string -- SVG image alternative text (* mandatory if svgName is set)
Events for c3m-tabs
Name Parameters Description
changeTab event.originalEvent: Click event
event.index: Index of the current selected tab or dragged tab 		Callback to invoke on tab change
closeTab event.originalEvent: Click event
event.index: Index of the current deleted tab 		Callback to invoke on tab close

* mandatory properties
