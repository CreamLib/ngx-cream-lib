Full Modal

If actions inside the modal should close it, you can call the ToggleOpen function using the modal's identifier.

Of course, you may add aditional functions to the same event.

Label of the button

Content of the modal

 
<c3m-modal-trigger [modal]="exempleModal">
  Label of the button
</c3m-modal-trigger>

<c3m-modal #exempleModal modalTitle="This is the title of the modal">
  <p>Content of the modal</p>
</c3m-modal>

Dialog Modal

Opens small modal

Content of the modal

 
<c3m-modal-button [modal]="exempleModal2">
  Opens small modal
</c3m-modal-trigger>
<c3m-modal class="simple-dialog" #exempleModal2 modalTitle="Cancel contract">
  <p>Content of the modal</p>
  <p class="buttons-bar">
    <button (click)="test();exempleModal2.ToggleOpen()">Ok</button>
  </p>
</c3m-modal>

Properties and Events

Properties applicable to the component c3m-modal
Name Type Default Description
modalTitle * string -- Title of the modal
<identifier> * #xxxxx -- Unique identifier of the modal
simple-dialog (class) -- Displays modal in small format, for quick feedback message
Properties applicable to the component c3m-modal-button
Name Type Default Description
[modal] * string -- Value of the target modal unique identifier
disabled string -- Disables button

* mandatory properties
