Dropdown with label

<c3m-dropdown label="Label of dropdown">
  <c3m-dropdown-item>
    <NG-CONTENT>
  </c3m-dropdown-item>
</c3m-dropdown>

ng-content must be a link or a button only

Dropdown with icon

<c3m-dropdown imgSrc="nav-feedbacks" imgAlt="Describe the nature of the actions">
  <c3m-dropdown-item>
    <NG-CONTENT>
  </c3m-dropdown-item>
</c3m-dropdown>

Dropdown "Other Actions"

In this case, it is mandatory to add a altLabel value to inform the nature of the actions in the dropdown. 

<c3m-dropdown class="others" ariaLabel="Describe the nature of the actions">
  <c3m-dropdown-item>
    <NG-CONTENT>
  </c3m-dropdown-item>
</c3m-dropdown>

Properties and Events

Properties applicable to the component c3m-dropdown
Name Type Default Description
label string "" Defines label for dropdown trigger button
imgSrc string "" Replaces label by the image indicated by its Id (should be listed in "/assets/img/sprite.svg").
imgAlt string "" Textual alternative to image. Mandatory if imgSrc is set.
others (class) -- Replaces label and arrow by a "kebab" icon

* mandatory properties
