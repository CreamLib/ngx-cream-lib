Properties and Events

Properties applicable to the component c3m-accordion
Name Type Default Description
titleLevel string "2" Level of title for the accordion blocks; use it to adjust the component to the document's hierarchy
label string "" Adds link with given absolute URL to chip's label
Properties applicable to the component c3m-accordion-item
Name Type Default Description
label string "" Defines the title of accordion item
[isOpen] boolean false Indicates if accordion item should be opened on page load
total string "" Allows to indicate a total value for the accordion item content
