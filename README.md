Installation
=====================

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.0.4.

Run
`npm install ngx-cream-lib` 

In your app.module.ts
`import { NgxCreamLibModule } from 'ngx-cream-lib';`
And
`@NgModule({
  declarations: [
    NgxCreamLibModule
   ]
  })`
